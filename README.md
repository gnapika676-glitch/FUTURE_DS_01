 #Business Sales Performance Analytics Dashboard

## Overview

This dashboard analyzes sales data to uncover meaningful business insights and support data-driven decision-making.

It provides an interactive view of key business metrics, helping users evaluate sales performance across products, categories, sub-categories, and regions.

---

## Objectives

* Analyze overall business sales performance.
* Identify profitable and loss-making product segments.
* Compare regional performance.
* Track key metrics such as Sales, Profit, Quantity, and Discount.
* Enable interactive exploration through filters and slicers.

---

## Dashboard Features

* Interactive Power BI Dashboard
* Dynamic filtering using slicers
* Sales analysis
* Profit analysis
* Quantity analysis
* Discount analysis
* Category and Sub-Category performance analysis
* Product-level performance analysis
* Regional performance comparison
* User-friendly visualizations for decision-making

---

## Key Insights

* Copiers are the most profitable sub-category.
* Tables generate high sales but result in overall losses.
* The West region is the most profitable region.
* Product-level performance varies significantly across the business.

---

## Dataset Limitation

> **Note:** The dataset contains sales records only up to **January 5, 2018**. Therefore, data for 2018 is incomplete and should not be directly compared with previous full-year data.
